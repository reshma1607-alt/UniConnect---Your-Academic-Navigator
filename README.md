# UniConnect – Your Academic Navigator

UniConnect is a smart academic platform designed to help students prepare efficiently by providing important study materials, previous questions, repeated exam questions, notes, and interview preparation resources.

## 🚀 Features

- 📄 Previous Year Questions  
- 🔁 Frequently Repeated Exam Questions  
- ⭐ Important Questions for Exams  
- 📝 Study Notes for Different Subjects  
- 📚 Library Book Availability Tracker  
- 💼 Company-wise Interview Questions  
- 💡 Project Ideas and Resources  

## 🎯 Purpose

This platform helps students:
- Prepare better for exams  
- Focus on important and repeated questions  
- Access study materials in one place  
- Get ready for job interviews
